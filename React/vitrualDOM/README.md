# 🔥 Virtual DOM Demo

## 📌 What is Virtual DOM?

Virtual DOM is a lightweight JavaScript object that represents the real DOM.

React uses it to:

- Compare changes (diffing)
- Update only necessary parts
- Improve performance

---

## ⚙️ How It Works

1. State changes
2. New Virtual DOM is created
3. Compared with old Virtual DOM
4. Only changed parts update in real DOM

---

## 🧪 What to Observe

- Open browser console
- Click button
- Notice re-render logs
- Only text updates, not full UI

---

## 🎯 Key Learning

- React does NOT update full DOM
- It updates only what's needed
